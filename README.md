# minnan.org authentic homepage v1.4

Performance release based on the “Authentic Minnan — past, present and future” homepage.

## Performance changes
- Converted large PNG photography to compressed WebP.
- Preloaded the hero photograph.
- Lazy-loaded and asynchronously decoded below-the-fold images.
- Added image dimensions to reduce layout shift.
- Deferred off-screen section rendering with `content-visibility`.

Upload all files and folders to the repository root and replace existing files.
